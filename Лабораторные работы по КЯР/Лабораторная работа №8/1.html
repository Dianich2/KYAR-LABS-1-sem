Свойтво display позволяет изменять строчные элементы на блочные или наоборот, или делать их строчно-блочными. Свойство border-radius, позволяющее добавлять скругления к одному или нескольким углам элемента. Для добавления теней к блоку, обрамляющему элемент, используется свойство box-shadow.

Для изменения отступа от границ до содержимого ячеек используется свойство padding. Чтобы настроить расположение содержимого внутри ячейки, используются свойства text-align для выравнивания по горизонтали и vertical-align для выравнивания по вертикали. Существует линейный и радиальный градиенты. background-image: linear-gradient() позволяет задать линейный градиент. background-image: radial-gradient() позволяет задать радиальный градиент. Плавный переход между цветами можно убрать с помощью следующего кода: background: repeating-linear-gradient(). C помощью свойства background-blend-mode можно применять режимы смешивания.

Преобразование осуществляется с использованием свойства transform. Оно имеет такие значения: 1.scale() — увеличить или уменьшить в размерах элемент; 2. rotate(deg) — повернуть элемент на определенный угол заданный в deg; 3. translate(px, px) — смещение элемента из его текущей позиции на некоторое расстояние; 4. skew(deg) — изменяет форму элемента, смещая верхний край элемента в одну сторону, а нижний — в противоположную. C помощью свойства transformorigin можно изменить точку трансформации. Чтобы свойства работали при наведении на объект, нужно использовать псевдокласс :hover.

Переход представляет собой анимацию смены одного набора свойств CSS другим за определенный промежуток времени. Переходы задаются с помощью свойства transition. Первым указывается значение свойства transition-property, которое определяет для каких свойств должен присутствовать переход. Вторым следует свойство transition-duration, которое указывает продолжительность перехода к конечному значению. Для замедления или ускорения перехода после значения продолжительности используется свойство transition-timingfunction. Для задержки перехода последним устанавливается значение свойства transition-delay
